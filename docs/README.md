# lobster-bot Documentation

This directory contains internal reference documentation for lobster-bot. It is written for Claude to read when it needs to understand how the bot works, what features are available, and how to extend it.

## Files

- **architecture.md** — How the bot works end to end: message flow, process model, file structure, how Claude Code is invoked.
- **features.md** — Every feature documented: commands, media handling, memory, sessions, hooks, agents, cost tracking.
- **customization.md** — How to make the bot yours: personality, config, tools, agents, custom commands, hooks, MCP servers, memory.
- **deployment.md** — How to deploy and maintain: system requirements, systemd, autosync, environment variables, logs, debugging.
